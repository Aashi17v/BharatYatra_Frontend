# 🌍 Tour & Travel Website

A modern and responsive travel website built using Next.js.
This project showcases popular destinations with an interactive UI and smooth navigation experience.

---

## ✨ Features

* 🏝️ Multiple destination pages (Delhi, Mumbai, Kerala, Goa, etc.)
* 🔗 Dynamic routing for detailed destination pages
* 📱 Fully responsive design (mobile + desktop)
* 🎨 Clean and modern UI using SCSS
* ⚡ Fast performance with Next.js

---

## 🛠️ Tech Stack

* Next.js
* React.js
* JavaScript
* SCSS (Sass)

---

## 📁 Project Type

Frontend Only (No Backend / No Database)

---

## 🚀 Getting Started (Run Locally)

1. Clone the repository

```
git clone https://github.com/Aashi17v/BharatYatra_Frontend.git
```

2. Navigate to project folder

```
cd BharatYatra_Frontend
```

3. Install dependencies

```
npm install
```

4. Run development server

```
npm run dev
```

5. Open in browser

```
http://localhost:3000
```
